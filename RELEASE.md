
# Инструкция по релизу

## Обновление версии игры

```bash
node tools/bumpVersion.js X.Y.Z
```

Проверить локально:
- Двойной клик по «Запустить игру.exe»
- Нажать Ctrl+F5
- Проверить, что в углу vX.Y.Z
- Проверить, что консоль без ошибок

Затем:
```bash
git add . && git commit -m "Release X.Y.Z" && git push
```

Через ~10 минут открыть сайт (можно с телефона) — в углу должна быть новая версия.

## Архив для заказчика

Для создания архива для заказчика:
1. Убедиться, что «Запустить игру.exe» есть в корне проекта
2. Скопировать всю папку игры, исключив:
   - .git/
   - tools/
   - любые резервные копии
3. Создать ZIP-архив из оставшихся файлов
4. При отправке предупредить:
   > Windows SmartScreen при первом запуске может спросить подтверждение — нажмите «Подробнее → Выполнить в любом случае».
